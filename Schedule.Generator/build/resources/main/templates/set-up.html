Default Title

Roster

This is a list of all current employees. You can add or remove employees. If you click on the last name of an employee it will take you to their profile. There you can view or edit their schedule and training levels.

Training List

This is a list of all of the training types. Each training has a rarity so that the posts that require the most specialized training get priority. You can add or remove training types. If you click on a training's name it will take you to a roster of the employees with that training.

Post List

This is a list of all the posts. They are organized by which training types they require. Each post also has a priority level to make sure that the most needed posts get posted first when we are short staffed. You can add or delete posts here.

Shifts

This is a list of each shift in the course of the week. If you click on a shift, it will pull up a roster of the employees that work that shift.
